# Contribution
- make sure you have
    - [python 3.11.*](https://www.python.org/downloads/) installed
    - [poetry](https://python-poetry.org/docs/) installed

- Clone the repo
- setup python environment
    - ```
        poetry install
        ```
- activate the environment
- install pre-commit hooks
    - ```
        pre-commit install
        ```
